Overview

The Stripe Payment API provides a comprehensive solution for processing payments online. Whether you're building an e-commerce platform, SaaS application, or marketplace, Stripe offers the tools you need to accept payments securely.

With support for over 135 currencies and multiple payment methods including credit cards, digital wallets, and bank transfers, Stripe makes it easy to accept payments from customers worldwide.

Quick Start

JavaScript 
// Install Stripe SDK
npm install stripe

// Initialize Stripe
const stripe = require('stripe')('sk_test_...');

// Create a payment intent
const paymentIntent = await stripe.paymentIntents.create({
  amount: 2000,
  currency: 'usd',
  payment_method_types: ['card'],
});

console.log(paymentIntent);
